# Operation Bitcoin Mining Dashboard

A Hydrapool mining dashboard for Operation Bitcoin, based on [Hashdash](https://github.com/dplusplus1024/hashdash) by D++.

## Setup
Static HTML — open `index.html` in a browser or serve with any static file server.
